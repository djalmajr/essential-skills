# agile-sprint-review

Prepares sprint review and demo of deliveries for stakeholders.

## When to use

- When the sprint has ended and you need to present what was delivered
- When stakeholders need to see the result of deliveries
- When you need to validate that the product is on the right track
- To close the cycle between sprint planning and retrospective
- When scope changed and decisions are needed

## How to use

```
/agile-sprint-review
```

Example: `/agile-sprint-review sprint-12`

See [full documentation](../../docs/agile/agile-sprint-review.md) for end-to-end examples and tips.
