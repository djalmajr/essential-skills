# agile-sprint-planning

Plans sprint by selecting items from backlog, defining objective, capacity, and execution order.

## When to use

- At the beginning of a work cycle to align what will be done
- When you need to select backlog items based on value, risk, and dependency
- When recording capacity and constraints for a sprint
- When validating that each item has Definition of Ready

## How to use

```
/agile-sprint-planning
```

Example: `/agile-sprint-planning sprint-13`

See [full documentation](../../docs/agile/agile-sprint-planning.md) for end-to-end examples and tips.
