# agile-epic

Structures large initiatives into story backlog with roadmap, dependencies, and verification.

## When to use

- When work requires several coordinated stories
- When there are dependencies between deliveries that need coordination
- When a roadmap is needed to guide the sequence
- After a refinement that generated several stories
- When the initiative has size L or XL

## How to use

```
/agile-epic
```

Example: `/agile-epic auth-refactor`

See [full documentation](../../docs/agile/agile-epic.md) for end-to-end examples and tips.
