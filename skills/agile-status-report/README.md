# agile-status-report

Consolidates status of a period or milestone with deliveries, deviations, risks, and next steps.

## When to use

- When you need a consolidated view of an initiative's progress
- When you need to report on deliveries, deviations, and risks from a period
- When you need to distinguish completed, in progress, and at risk items
- When a milestone or period needs summarization

## How to use

```
/agile-status-report
```

Example: `/agile-status-report auth-epic`

See [full documentation](../../docs/agile/agile-status-report.md) for end-to-end examples and tips.
