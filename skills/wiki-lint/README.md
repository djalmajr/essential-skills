# wiki-lint

Verificação de saúde e manutenção da wiki — audit, verify, clean, and organize the knowledge base.

## When to use

- When you need to audit, verify, clean, or organize the knowledge base
- When checking for broken cross-references, orphan pages, or stale content
- When verifying frontmatter consistency and raw/wiki source alignment
- When checking for contradictions or missing links between wiki pages

## How to use

```
/wiki-lint
```

Example: `/wiki-lint`

See [full documentation](../../docs/wiki/wiki-lint.md) for end-to-end examples and tips.
