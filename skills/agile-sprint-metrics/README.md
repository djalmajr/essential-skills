# agile-sprint-metrics

Consolidates objective metrics of a sprint including deliveries, blockers, deviations, and velocity.

## When to use

- When you need quantitative data about deliveries, blockers, deviations, and velocity
- At the end of a sprint, before review or retro
- When the team needs data to discuss performance
- To compare sprints and identify trends
- When there is doubt if declared capacity is calibrated

## How to use

```
/agile-sprint-metrics
```

Example: `/agile-sprint-metrics sprint-12`

See [full documentation](../../docs/agile/agile-sprint-metrics.md) for end-to-end examples and tips.
