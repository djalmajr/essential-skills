# agile-ceremonies-router

Orchestrates light Scrum ceremonies by deciding between refinement, sprint planning, or retrospective.

## When to use

- When you need to decide between refinement, sprint planning, or retrospective
- When you need guidance on which ceremony to conduct
- When you have a large or ambiguous backlog item (refinement)
- When starting a work cycle (sprint planning)
- When finishing a cycle or delivery (retrospective)

## How to use

```
/agile-ceremonies-router
```

Example: `/agile-ceremonies-router`

See [full documentation](../../docs/agile/agile-ceremonies-router.md) for end-to-end examples and tips.
