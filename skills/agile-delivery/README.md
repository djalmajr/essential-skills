# agile-delivery

Orchestrates delivery tracking by deciding between daily, status report, or post-implementation report.

## When to use

- When you need to decide between daily, status report, or post-implementation report
- When you need guidance on how to track a delivery
- When you're unsure which tracking artifact fits the current moment

## How to use

```
/agile-delivery
```

Example: `/agile-delivery sprint-12`

See [full documentation](../../docs/agile/agile-delivery.md) for end-to-end examples and tips.
