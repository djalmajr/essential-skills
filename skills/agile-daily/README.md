# agile-daily

Generates daily status update with progress, blockers, and next step.

## When to use

- When you need to record what advanced, what is blocked, and what the next observable step is
- When doing a daily standup or status check during a delivery cycle
- When a critical blocker needs to be escalated or a plan needs adjustment
- When delivery is close to closing and you want to transition to post-impl

## How to use

```
/agile-daily
```

Example: `/agile-daily auth-refactor`

See [full documentation](../../docs/agile/agile-daily.md) for end-to-end examples and tips.
