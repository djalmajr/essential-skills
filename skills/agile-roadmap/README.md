# agile-roadmap

Structures quarterly roadmaps and roadmaps by epic.

## When to use

- When you need to organize direction, priorities, phases, and dependencies
- When you need to connect roadmap items to epics and stories
- When building a quarterly roadmap for direction alignment
- When building a roadmap by epic for delivery sequencing

## How to use

```
/agile-roadmap
```

Example: `/agile-roadmap Q2-2026`

See [full documentation](../../docs/agile/agile-roadmap.md) for end-to-end examples and tips.
