# agile-proto

Create interactive UI prototypes with Bun + React + shadcn/ui + Tailwind v4.

## When to use

- When asked to "prototype", "create proto", or "mockup screens"
- When exploring UI flows before implementation
- When you need an interactive prototype to validate a design
- When you want to rapidly build a standalone UI demo

## How to use

```
/agile-proto
```

Example: `/agile-proto login-flow`

See [full documentation](../../docs/agile/agile-proto.md) for end-to-end examples and tips.
