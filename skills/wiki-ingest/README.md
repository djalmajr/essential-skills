# wiki-ingest

Processa uma nova fonte de raw/ para a wiki, criando ou atualizando páginas de conhecimento.

## When to use

- When you need to ingest, process, add, or incorporate a source into the knowledge base
- When a new raw source (transcript, document, PDF) needs to be processed into wiki pages
- When re-ingesting an already cataloged source to update the wiki

## How to use

```
/wiki-ingest
```

Example: `/wiki-ingest raw/meetings/2026-04-11-planning.md`

See [full documentation](../../docs/wiki/wiki-ingest.md) for end-to-end examples and tips.
