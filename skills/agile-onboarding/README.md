# agile-onboarding

Onboarding guide for new team members in the agile flow with AI.

## When to use

- When someone new joins the team and needs to understand the planning, execution, and tracking flow
- When someone changes roles (e.g., dev becomes tech lead)
- When the team adopts the flow for the first time
- When someone needs retraining after time away

## How to use

```
/agile-onboarding
```

Example: `/agile-onboarding new-dev`

See [full documentation](../../docs/agile/agile-onboarding.md) for end-to-end examples and tips.
