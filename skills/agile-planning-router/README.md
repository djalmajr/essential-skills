# agile-planning-router

Orchestrates decision between simple plan, story, or epic based on work size and clarity.

## When to use

- When you need to decide which planning artifact is appropriate for a demand
- When you need guidance on the planning flow
- When you're unsure whether to use plan, story, or epic
- When the problem is not defined and may need intake or refinement first

## How to use

```
/agile-planning-router
```

Example: `/agile-planning-router redesign-user-profile`

See [full documentation](../../docs/agile/agile-planning-router.md) for end-to-end examples and tips.
