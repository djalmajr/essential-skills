# wiki-query

Responde perguntas sobre a base de conhecimento da wiki, synthesizing answers from existing pages.

## When to use

- When asking about concepts, processes, entities, or any content from the wiki
- When you need an answer based on the knowledge base rather than re-synthesizing from scratch
- When you want to discover if information exists or identify gaps in the wiki

## How to use

```
/wiki-query
```

Example: `/wiki-query how does authentication work?`

See [full documentation](../../docs/wiki/wiki-query.md) for end-to-end examples and tips.
