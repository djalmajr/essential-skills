# agile-plan

Creates simple execution plan for localized changes.

## When to use

- When the work is small (XS or S)
- When work involves few files and can be executed in a single cycle
- When a story is already detailed and needs an operational plan
- When the change is localized and straightforward

## How to use

```
/agile-plan
```

Example: `/agile-plan add-button-component`

See [full documentation](../../docs/agile/agile-plan.md) for end-to-end examples and tips.
